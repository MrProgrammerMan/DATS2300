# [Kap 3.3.2]([[20250818102829]]) En lenket liste
En **enkeltlenket liste** består av noder som hver inneholder en verdi og en referanse til neste node. Klassen `Node` defineres som en **statisk, privat og generisk** indre klasse for å holde data internt i listeklassen. \
Fordeler med enkeltlenket liste:
* Mindre minnebruk enn dobbeltlenket liste
* Enklere implementasjon

## Grunnstruktur
```java
public class EnkeltLenketListe<T> implements Liste<T> {

    private static final class Node<T> {
        private T verdi;
        private Node<T> neste;

        private Node(T verdi, Node<T> neste) {
            this.verdi = verdi;
            this.neste = neste;
        }
    }

    private Node<T> hode, hale;
    private int antall;

    public EnkeltLenketListe() {
        hode = hale = null;
        antall = 0;
    }
}
```

## Innlegging bakerst (`leggInn(T verdi)`)
Legger til et element bakerst. Hvis listen er tom, peker både `hode` og `hale` til samme node.
```java
public boolean leggInn(T verdi) {
    Objects.requireNonNull(verdi, "Ikke tillatt med null-verdier!");
    if (antall == 0) hode = hale = new Node<>(verdi, null);
    else hale = hale.neste = new Node<>(verdi, null);
    antall++;
    return true;
}
```

## Innlegging på gitt indeks (`leggInn(int indeks, T verdi)`)
Setter inn verdien på en spesifikk plass.
* `indeks == 0`: foran
* `indeks == antall`: bakerst
* ellers: midt i listen
```java
public void leggInn(int indeks, T verdi) {
    Objects.requireNonNull(verdi, "Ikke tillatt med null-verdier!");
    indeksKontroll(indeks, true);

    if (indeks == 0) {
        hode = new Node<>(verdi, hode);
        if (antall == 0) hale = hode;
    } 
    else if (indeks == antall) {
        hale = hale.neste = new Node<>(verdi, null);
    } 
    else {
        Node<T> p = hode;
        for (int i = 1; i < indeks; i++) p = p.neste;
        p.neste = new Node<>(verdi, p.neste);
    }
    antall++;
}
```

## Viktige konsepter
* **`hode`** peker til første node i listen
* **`hale`** peker til siste node, gir rask tilgang ved innsetting bakerst
* **`antall`** holder styr på hvor mange noder som finnes
* **Statisk indre klasse (`Node`)** betyr at den ikke avhenger av en instans av den ytre klassen
* **`Objects.requireNonNull()`** brukes for å forhindre `null`-verdier

## Oppsummering
En enkeltlenket liste med indre noder er en effektiv og minnebesparende implementasjon.\
Ved å bruke både `hode` og `hale` oppnår man **O(1)** innsetting bakerst.\
Metodene kan utvides til å støtte fjerning, søk og iterasjon basert på samme prinsipp.
