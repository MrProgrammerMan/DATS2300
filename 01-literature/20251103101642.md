# [Kap 4.1.1]([[20250818102829]]) Hva er en stack?
En stack er en datastruktur der vi alltid legger inn ny data på toppen, og tar ut fra toppen. Konseptuelt ligner det en stabel med ting. \
Kalles også for **Last-In-First-Out Queue(LIFO)**.
I bunn og grunn trenger vi bare å legge inn og ta ut gjenstander fra en stack. Likevel er det vanlig å implementere funksjonalitet for å kunne:
- Se antallet ting i stacken
- Kikke på neste element uten å ta det ut
- Sjekke om stacken er tom
- Nullstille stacken(tømme den)
Eller annen nyttig funksjonalitet.
```java
public interface Stack<T> {
    public void push(T value);
    public T peek();
    public T pop();
    public int size();
    public boolean isEmpty();
	public void clear();
}
```
