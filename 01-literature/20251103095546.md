# [Kap 3.2.4]([[20250818102829]]) En indre iteratorklasse
Det er ofte nyttig for listestrukturer å kunne itereres gjennom. En generell og abstrakt måte å gjøre dette (fremfor eksempelvis å telle gjennom indeksene til lista) er å implementere en iterator. Dersom vi implementerer en iterator som en indre klasse i listen kan vi benytte klassevariablene i listen.
```java
private class TabellListeIterator implements Iterator<T> {
    private int denne = 0;
    public boolean hasNext() {
    	return denne < antall;
    }
    public T next() {
        if (!hasNext()) throw new NoSuchElementException("Tomt eller ingen verdier igjen!");
        return a[denne++];
    }
}
```
Iteratoren går kun gjennom listen ved å telle, fordi den er opptatt av den indre strukturen til lista. En som benytter iterator trenger ikke denne kunnskapen, og klassen vil fungere bedre med generiske metoder. Denne abstraksjonen er viktig i programmering.
