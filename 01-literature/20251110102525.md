# [4.2.2]([[20250818102829]]) En sirkulær kø
En kø kan benytte et array som indre datastruktur, men dette krever en smule tenking angående hvordan dataen skal bevege seg. Vi ønsker ikke å flytte alle elementene ettersom vi tar ut av køen. Derfor er vi nødt til å holde styr på hvor i arrayet vi er til enhver tid. Se for deg at køen på matbutikken står stille mens den ansatte går fra person til person; totalt sett krever det mindre bevegelse. Det mest kompliserte er å utvide arrayet ved behov, men det er ikke spesielt kostbart.
```java
public class ArrayQueue<T> implements Queue<T> {
	private T[] a;
    private int from;
    private int to;

    @SuppressWarnings("unchecked")
    public ArrayQueue(int size) {
        if (size < 1) throw new IllegalArgumentException("Cannot create an arrayqueue with size < 1");
        a = (T[]) new Object[size];
        from = to = 0;
    }

    public ArrayQueue() {
        this(8);
    }

    @SuppressWarnings("unchecked")
    private void extend() {
        int len = a.length;
        a = Arrays.copyOf(a, len * 2);
        if (from == 0) {
            to = len;
            return;
        }
        System.arraycopy(a, from, a, from + len, len - from);
        from += len;
    }
    
    public boolean add(T value) {
        a[to++] = value;
        if(to == a.length) to = 0;
        if (to == from) extend();
    }
    
    public T peek() {
        if (isEmpty()) throw new NoSuchElementException("The queue is empty");
        return a[from];
    }
    
    public T remove() {
        if (isEmpty()) throw new NoSuchElementException("The queue is empty");
        T val = a[from];
        a[from++] = null;
        if (from == a.length) from = 0;
		return val;
    }
    
    public int size() {
        if (to >= from) {
            return to - from;
        } else {
            return a.length - from + to;
        }
    }
    
    public boolean isEmpty() {
        return to == from;
    }

    @SuppressWarnings("unchecked")
    public void clear() {
        from = to = 0;
        a = (T[]) new Object[a.length];
    }
}
```
